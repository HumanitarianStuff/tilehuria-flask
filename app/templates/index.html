Please upload a polygon file or draw a rectangle


Do you want to select the boundaries of the tiles by GeoJSON or by drawing a rectangle?

It should be a GeoJSON file containing one or more polygons, in WGS84 Coordinate Reference System (this should be the default for most tools that produce such files).

With a bit of luck, after submitting your file you will find a corresponding MBTile file on the MBTiles page.


Use the square button on the top right corner of the map to draw a rectangle. You can edit or delete the rectangle by using the other top right buttons. Then you can continue to fill the form on the right
